Pesquisa de endereço pelo CEP

ViaCEP é uma API que faz uma consulta com o cep digitado e pode ser retornado os seguintes tipos "json", "xml", "piped" ou "querty". Esse primeiro método usamos para fazer uma consulta usando o CEP para retornar bairro, rua, cidade e estado referentes aquele CEP. No segundo método utilzamos a consulta ao contrario, dessa vez digitamos a UF do estado, cidade e rua, pode ser o nome completo da rua ou so uma parte, neste caso retornará todas a ruas com aquele nome.
